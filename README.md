# CryptCode Web Demo

A small Angular site built for our CS 420 CryptCode project.

CryptCode is a custom interpreted language where readable `.ccode` files can be encrypted into `.crypt` files and then run through a matching interpreter. The site gives a quick overview of the language, shows a few sample programs, and includes a browser-based test drive where you can write CryptCode, encrypt it, and see the output.

## Live site

https://rlopez4042.github.io/cryptcode-web/

## What is included

- About page explaining what CryptCode is
- Demo page with sample `.ccode`, `.crypt`, and output views
- Test Drive page for writing and running CryptCode in the browser
- Mobile friendly layout
- Links to both the language repo and website repo
