Crypt Code

Encrypted custom programming language

CryptCode is a custom programming language where the readable source code is transformed into an encrypted .crypt file. The key tells the interpreter how to decrypt each line, rebuild the program logic, and display the output.

What is CryptCode?

CryptCode is a small interpreted language built around the idea that source code can be stored encrypted instead of being saved as plain readable text.

How do you use it?

You write a plain .ccode file using CryptCode keywords, generate an encrypted .crypt file, then run the encrypted file with a key.

Core syntax

  • print <message or variable>
  • count <var> from <start> to <end>
  • when <var> mod <number> is <number>
  • else
  • end

The basic flow

  1. Create a readable .ccode file.
  2. Encrypt each line using the selected encryption methods.
  3. Save the result as a .crypt file.
  4. Run the encrypted file using the key.
  5. Decrypt the lines, rebuild the logic, and display the output.
